Addendum

  • Problem Statement: This project was designed to create a webpage for generating random quotes.
  • Solution Overview: HTML, CSS, and JavaScript were employed to build and implement the Random Quote Generator's functionality.
  • Implementation Details: HTML was used to structure the page, organizing the header, navigation, main content area for displaying quotes, sidebar with external links, and footer. CSS provided styling for the layout and ensured consistency and responsiveness across elements. JavaScript was used to fetch and display quotes, update the last modified date in the footer, and manage user actions, like clicking the "New Quote" button.
  • Challenges Encountered: Some of the challenges involved making sure the site was compatible with multiple browsers and responsive across various devices and screen sizes. Additionally, dynamically loading and displaying quotes required either connecting to an external API or creating a database.
  • Resolution Strategy: By utilizing modern web development practices, we addressed these challenges effectively. HTML, CSS, and JavaScript were combined to deliver a Random Quote Generator that meets user needs.
